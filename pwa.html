Progressive Web Apps

?

Que es lo que en realidad resuelve una app?
Basicamente, quisieramos deshacernos de este intermediario. Quisieramos poder aprovechar el código y las utilidades que ya creamos para el sitio web Quisieramos poder acceder a las utilidades del telefono Quisieramos poder acceder a la aplicación offline Quisieramos poder tener push notifications
Los web browsers ya pueden hacer todas estas cosas
Una pwa es una aplicación que utiliza tecnologías web, pero que se ve y se siente como una aplicación mobil
No necesitan aprender un nuevo lenguaje o un nuevo framework, ni siquiera instalar nada nuevo
https://appetiser.com.au/wp-content/uploads/2019/02/Progressive-Web-Apps.png
Sin embargo, tomen nota de que esto es tecnología nueva. Su soporte de navegadores no es perfecto y las especificaciones aún cambian constantemente.

Service Workers

Los service workers proveen funcionalidad offline y de background, entre otras como push notifications.

Funcionan en base a eventos, que escuchan en diferentes etapas de su ciclo de vida

Registrando un service worker


if ('serviceWorker' in navigator) {
  window.addEventListener('load', function() {
    navigator.serviceWorker.register('/sw.js').then(function(registration) {
      // Registration was successful
      console.log('ServiceWorker registration successful with scope: ', registration.scope);
    }, function(err) {
      // registration failed :(
      console.log('ServiceWorker registration failed: ', err);
    });
  });
}
Devtools > Application > ServiceWorkers

const urlsToCache = [
  '/',
  '/css/materialize.min.css',
  '/js/materialize.min.js'
]

self.addEventListener('install', function(event) {
  event.waitUntil(
    caches.open('my-first-cache-v1')
      .then(function(cache) {
        console.log('Opened cache');
        return cache.addAll(urlsToCache);
      })
  )
})



self.addEventListener('fetch', function(event) {
  event.respondWith(
    caches.match(event.request)
      .then(function(response) {
        // Cache hit - return response
        if (response) {
          return response;
        }
        return fetch(event.request);
      }
    )
  )
})
