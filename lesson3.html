CSS

Cascading Style Sheets

  • Se encarga de la presentación del contenido
  • Se puede utilizar para estilizar cualquier documento XML (incluyendo HTML y SVG)
  • Fue diseñado para separar la presentación y el contenido

Selector

index.html


              <h1>Hello World</h1>

index.css


              h1 {
                color: Maroon;
              }

Class Selector

index.html


              <h1 class="title">Hello World</h1>

index.css


              h1.title {
                color: Maroon;
              }

ID Selector

index.html


              <h1 class="title" id="num4">Hello World</h1>

index.css


              h1.title#num4 {
                color: Maroon;
              }

Multiple Selector

index.html


              <h1 class="title" id="num4">Hello</h1>
              <h1 class="sub" id="num3">World</h1>

index.css


              .title, .sub {
                color: Maroon;
              }

Colores en CSS

por nombre

                color: Maroon;
              }

por hexadecimal


              .title {
                color: #0CF;
                color: #00CCFF;
                color: #00CCFF7F;  /* html 6? */
              }
https://drafts.csswg.org/css-color/#hex-notation00CCFF

              .title {
                color: rgb(122, 133, 0);
                color: rgba(122, 133, 0, 0.25);
              }

por hsl


              .title {
                color: hsl(90, 50%, 10%);

              }

Fuentes en CSS

Cambiando la fuente de todos los elementos

                }

Recursos locales de fuentes


                @font-face {
                    font-family: myFirstFont;

                }

Recursos remotos de fuentes


                <link href="//fonts.googleapis.com/css?family=Open+Sans" />
