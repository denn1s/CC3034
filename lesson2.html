Resumen de DNS

  1. Revisión local del dominio (/etc/hosts o cache)
  2. Solicitúd al servidor de DNS (8.8.8.8, 8.8.4.4)
  3. El servidor de DNS revisa si ya conoce el TLD, si no, se le solicita al root server
  4. Se le solicita al TLD (.com) la IP del servidor web
  5. Se le solicita al servidor web el subdominio (virtual host)
  6. Se le solicita al virtual host el recurso
