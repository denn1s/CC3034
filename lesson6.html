Peculiaridades de trabajar en javascript del lado del servidor

Asynchronous Programming

Callbacks


              function doThis(doThisOtherThing) {
                // do some stuff
                doThisOtherThing()
              }

Ejemplo: JQuery GET


              $.get(function (data) {
                // do stuff with data
              })
              doOtherStuff()  // this will be called before the callback! (probably)

Promises


              deThis().then(doThisOtherThing)

Ejemplo de declaración de promesas: ES6


              function doThis() {
                return new Promise(function(resolve, reject) {
                  // do some stuff
                  if (allOk) {
                    resolve()
                  } else {
                    reject()
                  }
                })
              }

Events


              myObject.on('someEvent', () => {
                console.log('an event occurred!');
              });
              // ...
              myObject.emit('event');

Ejemplo EventEmmiter


              const EventEmitter = require('events');

              class MyEmitter extends EventEmitter {}

              const myEmitter = new MyEmitter();
              myEmitter.on('event', () => {
                console.log('an event occurred!');
              });
              myEmitter.emit('event');
