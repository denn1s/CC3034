Javascript transpiler

source-to-source compiler

  • Los navegadores y engines server side sólo soportan javascript
  • De hecho, ni siquiera soportan cualquier versíon de javascript
  • https://kangax.github.io/compat-table/es6/
  • Aunque no soporten las especificaciones nuevas de js, al final los navegadores solo soportan lo que pueden ejecutar
  • ¿Entonces, por qué?
  • Comodidad de trabajo
  • Ordenar mejor el código
  • Permiten utilizar las funciones nuevas de javascript
  • Disminuye la complejidad

Coffeescript

  • Ruby, Python, Haskell...
  • Sintaxis basada en tabs y no en {}
  • Array comprehensions
  • Mantenido fuertemente por la comunidad de Ruby (viene incluído en RoR)

DEMO: Cofeescript

http://coffeescript.org

Typescript

  • Programacíon orientada a objetos (Java, C++)
  • Extiende javascript, es decir todo código de javascript corre en typescript
  • Agrega tipos y clases
  • Mantenido por Microsoft (incluído en visual studio)

DEMO: Typescript

https://www.typescriptlang.org
