NODE.JS

¿Qué es?

Utiliza V8 (chrome) para poder correr Javascript en sus computadoras (y en los servidores).

Permite que Javascript pueda hacer cosas que normalmente no puede, como servir http y acceder a archivos.

  • Javascript runtime
  • Chrome V8
  • Event-driven, non-blocking I/O (libuv)
  • Es un projecto diseñado para crear programas que hablan:
    • con redes
    • con filesystems
    • con I/O en general

¿Para qué lo usamos?

  • Para correr utilidades escritas en Javascript
  • Para correr servidores de javascript
  • Para conectar servicios web
